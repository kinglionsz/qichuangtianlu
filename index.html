赛道详情

路线概览

起点/终点：满京华艺术村
全程距离：131.4 km
总海拔爬升：3,121 m
预计耗时：8小时45分钟
难度评级：虐

赛道分段

  1. 葵涌启程 → 鹅公湾（沿海爬升）
  2. 同富 → 西涌（山海交界）
  3. 西涌 → 杨梅坑（海岸线疾驰）
  4. 杨梅坑 → 坝光（深入山径）
  5. 终极返程 → 满京华（意志考验）

行前准备

  • 确认具体出发日期
  • 检查天气预报（沿海风力与降雨）

  • 准备应急包（备用内胎、打气筒、工具组、急救包）

骑行路书

路书概述

基于行者路书 #1387571，2017 VAUDE 骑闯天路资格赛·深圳站完整路线。全程 131.4 公里，总爬升 3,121 米，途经大鹏半岛最美海岸线与最虐爬坡路段。

第一赛段：葵涌启程 → 鹅公湾（沿海爬升）

路线：满京华艺术村 → 葵鹏路|S360 → 迭福路 → S360 → 迎宾路 → 葵南路 → 水头路 → 坪西路|S359 → 海滨北路|S359 → 同富路 → 富民路

折返点：鹅公湾度假村入口

关键挑战：富民路往鹅公湾路段（一级虐点）

补给建议：能量胶×3，水750ml，香蕉×1 → 补水+盐丸，检查胎压

第二赛段：同富 → 西涌（山海交界）

路线：富民路 → 同富路 → 南西公路|S359

折返点：西涌观景台

关键挑战：西涌返南西公路（一级虐点）

补给建议：能量棒×1，电解质水500ml

第三赛段：西涌 → 杨梅坑（海岸线疾驰）

路线：南西公路|S359 → 新丰路 → 新大路|X260 → 新东路|S260

折返点：杨梅坑滨海度假中心公交站对面海滨观景台

补给建议：快速碳水（面包/能量饼），补水

第四赛段：杨梅坑 → 坝光（深入山径）

路线：新东路|S260 → 新大路|S260 → 葵南路 → 迎宾路 → S360 → 迭福路 → 葵鹏路|S360 → 金业大道 → 银葵路 → 葵坝公路 → 坝核公路

折返点：坝光坝核公路网大前网路口

关键挑战：葵坝路往坝光之径心水库路段（一级虐点）

补给建议：全面补给：水、电解质、能量胶×2

第五赛段：终极返程 → 满京华（意志考验）

路线：坝核公路 → 葵坝公路 → 银葵路 → 金业大道 → 葵鹏路|S360 → 满京华艺术村

关键挑战：葵坝路返葵涌之径心水库路段（一级虐点）

补给建议：终点补给：拉伸恢复，高蛋白补给
